# Privacy Policy
Modulus is a Discord moderation bot. This policy explains what data Modulus accesses and how it is used.
### Data Collected
Modulus collects only the minimum data required to function. This includes:
- Discord user IDs
- Discord guild IDs
- Moderation records such as warnings, timeouts, and bans
Modulus does not read or store private messages.
Modulus does not collect email addresses, IP addresses, or personal information outside of Discord identifiers.
### How Data Is Used
Collected data is used only to provide moderation features inside Discord servers.
Data is never sold, shared, or transferred to third parties.
### Data Storage
Moderation data is stored per server and only for as long as it is needed for moderation purposes.
Server owners may clear data at any time by removing the bot or resetting stored records.
### User Control
Server administrators control how Modulus is used in their server.
If you want moderation data removed, contact the bot developer.
### Changes
This Privacy Policy may be updated as Modulus evolves.
### Contact
For questions or data removal requests, contact the bot developer.
